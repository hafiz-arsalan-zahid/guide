# **App Name**: Ahmad's LifeHub

## Core Features:

- Todo List: Create and manage a personal to-do list with customizable categories and due dates.
- Marks Manager: Record and track your marks and grades for different tests and assignments.
- Subjects Manager: Organize your subjects with names and other data, such as a link to the subject's online resources.
- Timetable: Create and visualize a weekly timetable.
- Notes: Create, edit, and save text-based notes.
- AI Study Guide: AI-powered tool that provides relevant study suggestions for efficient exam preparation. The tool will incorporate external data, if required.
- Theme Switcher: Allow users to switch between a light and dark theme, adapting to different lighting conditions and user preferences.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), providing a sense of focus and calm for productivity.
- Background color: Light gray (#F5F5F5) for light mode, and darkest blue (#1A237E) for dark mode.
- Accent color: A muted purple (#9575CD), that is analogous to blue, used for interactive elements.
- Clean and readable sans-serif font for interface elements, ensuring readability.
- Use a set of modern, minimalistic icons to represent different tasks and tools within the app.
- Employ a responsive grid layout that adapts to different screen sizes and devices.
- Incorporate smooth transitions and animations for a more engaging user experience.
